# data-curtains

Exploring data curtains using Cesium.

Live demo available here:
[http://nasa-gibs.github.io/data-curtains/](http://nasa-gibs.github.io/data-curtains/)

## Quick Start

Install dependencies:
`npm install`

Run application:
`npm start`
